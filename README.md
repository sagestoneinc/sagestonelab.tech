# sagestonelab.tech

Marketing site landing page for SageStoneLab.

## Local preview

Open `index.html` directly in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Publish to GitHub Pages

This repository includes a Pages workflow at `.github/workflows/pages.yml`.

1. In GitHub, go to **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Merge changes to `main` (or run the workflow manually from **Actions**).
4. Your site will publish at `https://<org-or-user>.github.io/<repo>/` unless you configure a custom domain.
